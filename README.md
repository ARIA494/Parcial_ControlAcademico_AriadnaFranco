## Evidencias de consultas SQL

Se realizaron las siguientes consultas sobre la base de datos control_academico:

1. Mostrar todos los alumnos activos.
2. Mostrar los cursos con más de 3 créditos.
3. Mostrar los alumnos inscritos junto con el nombre del curso (JOIN).
4. Mostrar la cantidad de alumnos inscritos por curso (GROUP BY).

Estas consultas permiten analizar la información del sistema académico y verificar la correcta relación entre las tablas Alumno, Curso e Inscripcion.
